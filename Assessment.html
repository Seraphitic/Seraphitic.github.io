1 Algebraic Manipulation (1/27/21)

  1. Find \(x\) in each of the following equations
    1. \(2x = 30\)
    2. \(x - 15 = 7\)
    3. \(7x + 3 = x\)
    4. \(\frac{2x+1}{7}=5\)
  2. Find \(x\) and \(y\) for each of the systems of equations below
    1. \[-3x + 5y = -13\\ 2x + y = -26\]
    2. \[3x - 5y = -9\\ 5x + 2y = 16\]
    3. \[3x + 8y = -3\\ 2x + 5y = -3\]
  3. Divide the following polynomials by \((x+1)\)
    1. \(x^2 + 2x - 30\)
    2. \(x^3 + 1\)
    3. \(x^2 + 7x + 12\)
  4. Factor the following quadratics
    1. \(x^2 - 5x + 6\)
    2. \(x^2 - 2x - 8\)
  5. Simplify the following
    1. \(6x^2 - 8 + x^2 + 2x + 3\)
    2. \(\frac{2x - 5x^2 + 13 - 5x - 4x(x-1)}{4x}\)

2 Algebraic Manipulation II (2/3/21)

  1. Factorize the following expression
    1. \(9x - 3\)
    2. \(7x^2 + 4x\)
    3. \(x^2 - 2x - 15\)
    4. \(-2x^2 + 4x + 30\)
  2. Expand the following expression
    1. \(6x(3x + 4)\)
    2. \((3x-4)(5+x^2)\)
    3. \((1+x)^3\)
